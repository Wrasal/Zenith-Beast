# Описание
Гаджет - с помощью которого можно преодолевать расстояния горизонтальные и вертикальные.
Есть 2 типа:
- Статические - заранее заготовленный
- Динамический - устанавливается при помощи вторичного оружия
> [!grid]
> ![[Зиплайн.png]]

# Особенности
Персонаж цепляется за него во время падения или по клавише прыжка.
## Статический
- Заранее заготовленный стоит и ждёт использования.

## Динамический 
Снаряд для вторичного оружия. 
- Для использования надо подойти к заготовленным поверхностям
- Снаряд цепляется к определённым поверхностям
	- Пути карабканья
	- ... (Буду смотреть по лвл дизайну)
- Зип лайн остаётся, но можно сломать
  > [!grid]
  > ![[ДинамеческийЗипЛайн.gif]]
# Прогрессия
По началу игрок может использовать зип лайн для **спуска**. Далее по игре игрок получит предмет для подъёма.
