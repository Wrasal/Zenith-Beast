# `= this.file.name`
Вся игра разделена на идущие друг за другом миссии. Каждая миссия должна строиться по уникальному сценарию используя необходимые для своей реализации [[Системы]] согласно [[Прогрессии игрока]] по сюжету.

Сценарий миссии должен формироваться на основе жизненного цикла миссии приведенного ниже:
>### Жизненный цикл миссии
>[[Сюжетная вставка]] (открывающая) ---> Игровой процесc ---> [[Сюжетная вставка]] (закрывающая).

# Игровой процесс
Во время игрового процесса игрок может управлять [[Персонаж игрока|персонажем]] и взаимодействовать с [[Локация|локацией]].

>Во время игрового процесса могут быть включены [[Сюжетная вставка|сюжетные вставки]].

# Составляющие игрового процесса
Весь игровой процесс можно разделить на 3 основных составляющие:
- [[Изучение локации]]
- [[Подготовка локации]]
- [[Задание миссии|Выполнение задания миссии]]
- [[Боёвка]]

>Средняя продолжительность игрового процесса - **45–90 минут** без учета [[Сюжетная вставка|сюжетных вставок]].
