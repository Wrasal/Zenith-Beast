## Существуют с **испытаниями** : 
- Разрушенные базы
- Разрушение лабы 
- Старые механизмы

 Содержать испытание по прохождению которого будет доступно:
- Улучшения инвентаря (обвесы, гаджеты, способности)
- Изменения ландшафта
- Изменения квестов (упрощения\усложнение) 
  
 ## Без испытания
 
Нарративные приколы
(Придумать \ обсудить)
