# Ведение документа
Каждый текстовый файл внутри обсидиана условно назовем **документом**.

Все документы в проекте должны содержать в начале следующую текстовую конструкцию:

~~~mb
---
approved: false
---
~~~

Эти строки обозначают статус проверки.
- false - Не рассмотрено
- true - Одобрено

> Все **не рассмотренное** не принято в работу.  



# Цветовые обозначения
Если обьект выделен цветом отличным от дизайна обсидиана, то следуйте указаниями ниже:
- <span style="background:#ff4d4f">Текст</span> - Отменено. Информация отвергнута и больше не должна учитываться.
- <span style="background:#9254de">Текст</span> - На переработку. Информацию не отменена, но требует переработки.
- <span style="background:#d4b106">Текст</span> - Доработать. Информация принята, но требует более детального описания.
