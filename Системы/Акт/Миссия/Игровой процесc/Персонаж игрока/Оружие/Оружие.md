# Описание
Игрок может взять с собой 2 оружия:
- [[Основное оружие]]
	- Оружие дальнего боя. до ~100 м
- [[Вторичное оружие]]
	- Оружие среднего боя до ~15 м

# Виды

# Патроны и снаряды
Разновидность патронов даёт преимущества перед животными. Для каждого животного нужен индивидуальный подход и снаряды.
- Стандартные (для винтовки, дробь, болты, стрелы и т.п)
- Тяжелые - прокаченная версия некоторых стандартных патронов, для более толстокожих животных
- ... (нарезные)
