# Описание
Каждый элемент может комбинироваться. Стэлс путь может быть паркурным путём. Укрытие паркурным, cтэлс укрытие, cтэлс направитель и т.д.

# Элементы
- Стэлс 
- Паркурные 
- Укрытия
- Направители
  
## Стэлс
Пути на которых животное может нас не заметить (если правильно использовать его)

- **Углубления от корней** - рвы между корней, медленное передвижение, но незаметно
-  **КУСТЫ ПРАСТИ ГОСПАДЕ** - ну кусты ёпта, идёшь и прячешься, в близи животное замечает. Аналог плащ хамеллеон.
	- **Высотные кусты** - скопление веток и листвы на ветке дерева.
> (https://www.youtube.com/watch?v=QRX_baKV1DI - это я воюю против кустов)
- **Камни и рельеф** - камни которые расставлены как надо)

## Паркур
Пути для обхода врагов, разнообразия геймплея и нахождения новых выгодных позиции для охоты.

- Прыжки по веткам
- Карабканье
- Пути 

Могут служить укрытием или стэлс путём.
## Укрытия
Служат для быстрого сброса животного с хвоста и для защиты от него. Может сработать не всегда.

Укрытия могут находится в:
- Деревья
	- Корни
	- Базы на деревьях (постройки оставленные людьми )
- Скалах
- Подземные

Каждое укрытие визуально не приметное, но его выделяет мигающий огонёк на входе, есть 2 типа укрытия
- **Оставленное людьми** - на входе мигает лампочка, ели заметная
- **Природное** - на входе мигает мотылёк (или какое - ни будь другое насекомое)

## Направители
Визуальные штуки которые направляют игрока к
- Cюжетным миссиям
- Дополнительным миссиям 
- Интересным местам
- Выгодным позициям

Представляют из себя:
- Тропы - оставленные животными
- Постройки - расставленные так что бы привлечь внимание игрока
- 

В укрытиях могут находится ресурсы
# Интерактивность
- Батут
- [[Лазелка]]
Интерактивность - активация предмета с помощью гаджета
Каждый элемент можно сделать интерактивным. Когда элемент интерактивен он скрыт, сломан, не активен ...

Взаимодействовать можно с помощью:
- Оружия (выстрелить)
- Гаджетами (таких гаджетов ещё нет)
- Животными (помощь животного)

## Предложение (черновик)
Для первого акта подойдут интерактивная растительность
- Активационные паркурные пути для выгодных(тактичных) позициях (на этих позициях можно раскидать геймплейные и нарративные приколы)
	- Отстрелить мешающий объект
- Интерактивные растения
	- Скрытые проходы в корнях (стелс путь, приманка животных)
	- Расцветания дерева (паркур путь ,стелс путь, приманка животных)
	- Укрытия (активация защитной зоны)
