# Описание
Вся игра разделена на идущие друг за другом миссии. Каждая миссия должна строиться по уникальному сценарию используя необходимые для своей реализации системы согласно прогрессии игрока по сюжету.

Сценарий миссии должен формироваться на основе жизненного цикла миссии:
> ### Жизненный цикл миссии
> [[Сюжетная вставка]] (открывающая) ---> [[Игровой процесс]] ---> [[Сюжетная вставка]] (закрывающая).

Средняя длительность игрового процесса - **45–90 минут** без учета [[Сюжетная вставка|сюжетных вставок]].
