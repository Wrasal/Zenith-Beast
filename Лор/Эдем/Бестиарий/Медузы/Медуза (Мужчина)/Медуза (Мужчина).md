# Боевая роль

### Архетип
Штурмовик.

### Тактика
Действует по воле матки. Сближается с целью и атакует на короткой дистанции.

### Сильные стороны
- Быстрое сближение.
- Способность прерывать действия противника.
- Появляется непосредственно рядом с маткой.

### Слабые стороны
- Существенно слабее матки.
- Эффективен только на ближней дистанции.

## Передвижение
Перемещается по командам матки.

## Атаки и способности

### Рывок
Короткий бросок в сторону цели. Сбивает противника с ног.

### Удар током
Атака ближнего боя. Выпускает электрический разряд, вызывающий кратковременное прерывание действий цели.

## Особые свойства
Защищает матку при возникновении угрозы.
