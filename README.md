# Введение
Название игры: Zenith Beast  
Жанр: приключение, шутер от третьего лица, тактический.  
Платформы: PC  
Целевая аудитория: игроки одиночных атмосферных RPG и приключенческих шутеров.  
Краткое описание: Игрок берёт на себя роль авантюриста, исследующего чужую планету, где предстоит выживать и охотиться.
