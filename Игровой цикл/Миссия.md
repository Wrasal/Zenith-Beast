Миссия - определенная задача, которую может выполнить игрок.

Миссии делаться на типы:
- Контракт - Игрок обязан выполнить подобную миссию для прохождения уровня.
- Сюжетная - Игрок должен выполнить её чмайн квесты, двигают сюжет
- Дополнительные - не обязательные задания

Виды миссий:
- Изучение животных - раскрытие бестиария мира
- 

Скрытые активности - те которые игрок может получить по изучения мира** (На обдумывание)
