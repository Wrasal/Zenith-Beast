# Описание
Для прохождения [[Миссия|миссии]] игрок обязан выполнить поставленное ему задание миссии. Помимо задания миссии на [[Миссия|миссии]] присутствуют [[Дополнительные задания]].

# Типы заданий
Ниже приведен список заданий основанный на игровых [[Системы|системах]], которые могут использоваться как пазл для формирования итогового сценария на миссии.
Задачи:
1)    Изучение животных с мутациями 
- Воровство животных
- Изучений анализатором
- Убийство этого животного
- Изучение признаков присутствий
2)    Изучение точек интереса (загадки)
- Лабораторий (загадки с поведением животных: Приманкой подвести на кнопку и тп)
- Механизмов
- Местных достопримечательностей 
3)    Охота
- Охота с использованием гаджетов на животных

4)    Босс файты
- Долгая, насыщенная охота на животного(босса)
  
5) Стелс-гумманизм
- Незаметно пройти и забрать что - нибудь (убивай или не убивай)
