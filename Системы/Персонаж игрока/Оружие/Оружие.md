# Описание
Игрок может взять с собой 2 оружия:
- [[Основное оружие]]
	- Оружие дальнего боя. до ~400 м
- [[Вторичное оружие]]
	- Оружие среднего боя до ~30 м

# Виды
Оружие делится на виды
- **Основное оружие**
	- Винтовки
	- ...
- **Вторичное оружие**
	- Комбинированное ружье
	- ... (Луки)

# Подтипы
- [[Огнестрел]]
- [[Лазер]]
# Обвесы
На каждое оружие (с исключениями) можно повесить обвесы.
Обвесы - модули для оружие которые как то меняют её характеристики и свойства.
