## Типы снарядов
В арбалете 3 вида снарядов:
- ### Дротики - остаются после попадания
  Остаются при попадание на животном
	- _Метка_ - отображает животные на карте // можно отслеживать передвижение
	- _Транквилизаторы_ - на долго усыпляет цель
	- _Ультразвуковой хлопок_ - издает постоянный звук в течение n- секунд.  оглушает\пугает попавших под звучание. Радиус маленький
- ### Взрыв стрела 
   взрываются при прикосновение с поверхностью нанося минимальный урон. Каждый снаряд рассчитан на определённых животных // Может и наносят смотря какие животные будут
  - _Липкая жижа_ - покрывает место\участок в n метре жижей замедляя передвижение попавших в них, после n времени затвердевает и ломается, нанося урон
   - _Парализующий газ_ - задымляет участок, попавшие в него засыпают. При взаимодействие с ними просыпаются //заснуть могут не все животные, некий баланс на позднем этапе продумаем
   - _Ультразвуковой хлопок_ - Единоразово оглушает\пугает попавших под звучание. // Для сильно слышащих оглушает. 
  
- ### Бомбы - ловушки - устанавливаются при прикосновение с поверхностью
