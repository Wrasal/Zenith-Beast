# Ведение документа
Каждый текстовый файл внутри обсидиана условно назовем **документом**.

Все документы в проекте должны содержать в начале следующую текстовую конструкцию:

~~~mb
---
approved: false
---
~~~

Эти строки обозначают статус проверки.
- false - Не рассмотрено
- true - Одобрено

> Все **не рассмотренное** не принято в работу.

# Цветовые обозначения
Если обьект выделен цветом отличным от дизайна обсидиана, то следуйте указаниями ниже:
- <span style="background:#FF4D4F5E">Текст</span> - Отменено. Информация отвергнута и больше не должна учитываться.
- <span style="background:#9254de5E">Текст</span> - На переработку. Информацию не отменена, но требует переработки.
- <span style="background:#d4b1065E">Текст</span> - Доработать. Информация принята, но требует более детального описания.
- <span style="background:#0099995E">Текст</span> - Информация заморожена до лучших времен.
