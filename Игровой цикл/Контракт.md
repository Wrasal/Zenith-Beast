Контракт - Определенный сценарий в определённой части локации.

Контракты делаться на типы:
- Сюжетный - двигают сюжет
- Сторонний - дополнительный контент

Каждый контракт содержит задачи - то, что нужно сделать. задачи делаться на типы:
- Основная задача - игрок обязан выполнить подобную задачу для завершения контракта
- Дополнительные - не обязательные задачи (дополнительынй к)

Виды миссий:
- Изучение животных - раскрытие бестиария мира
- 

Скрытые активности - те которые игрок может получить по изучения мира** (На обдумывание)
