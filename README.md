# О проекте
**Жанр:** приключение, шутер от третьего лица, тактический, охота.
Платформы: PC
**Целевая аудитория:** любители одиночных атмосферных приключенческих шутеров.
**Краткое описание:** Игроку предстоит прожить историю путешествия группы авантюристов на другой планете.

# Ведение документа
Каждый текстовый файл внутри обсидиана условно назовем **документом**.

Все документы в проекте должны содержать в начале следующую текстовую конструкцию:

~~~mb
---
approved: false
---
~~~

Эти строки обозначают статус проверки.
- false - Не рассмотрено
- true - Одобрено

> Все **не рассмотренное** не принято в работу.
