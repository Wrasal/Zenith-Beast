# Описание
Особенные мобильности вызванные комбинациями базовых мобильностей.

## Элементы паркура
### Скольжение
При приседание во время бега начинает скольжение 
- Поддерживается скольжения если персонаж скользит со спуска (с горы)
### Перепрыгивание через препятствия 
При зажатой кнопки прыжка во время бега, персонаж будет перепрыгивать через препятствие
