# Описание
Карабканье в игре представлено в виде специальных отвесных стен [[Скаладром|скаладромов]].

# Функционал
[[Персонаж игрока]] может:
- Запрыгнуть на [[Скаладром|скаладром]] - (Персонаж во время прыжка, столкнувшись со [[Скаладром|скаладромом]] зацепляется за него переходя в режим карабканья)
- Карабкаться - (W/S двигается по вертикали, A/D двигается по горизонтали, комбинируется для перемещения по диагоналям)
- Спрыгнуть - (CTRL)
- Отпрыгнуть - (SPACE + W/A/S/D персонаж отпрыгивает от стены W - вверх, A/D - влево/вправо, S - назад за спину)
- Стрелять - пока заморожено





Например:
- Лианы
- Кора деревьев
- Скала с торчащими камнями
- ...

# Взаимодействие
Игрок может взаимодействовать со стеной:
- начать карабканье
	- Во время падения 
	- Вблизи нажать кнопку взаимодействия
- закончить карабканье
	- Отпрыгнуть от стены
	- Доползти до конца маршрута

## Особенности
**Управление** WASD не зависимо от камеры
Карабканье без стамины
На некоторые можно зацепиться с помощью [[Крюк кошка]]
Можно стрелять с одноручного оружия (пистолет)
