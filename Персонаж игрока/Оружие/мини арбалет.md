## Типы снарядов
В арбалете 3 вида снарядов:
- ### Дротики - остаются после попадания
  Остаются при попадание на животном
- ### Взрыв стрела 
   взрываются при прикосновение с поверхностью нанося минимальный урон. Каждый снаряд рассчитан на определённых животных // Может и наносят смотря какие животные будут
  - _Липкая жижа_ - покрывает место\участок в n метре жижей замедляя передвижение попавших в них, после n времени затвердевает и ломается, нанося урон
   - _Парализующий газ_ - задымляет участок, попавшие в него засыпают //заснуть могут не все животные, некий баланс на позднем этапе продумаем
   - _Ультразвуковой хлопок_ - оглушает попавших под звучание животных // для животных которые хорошо слышат
  
- ### Бомбы - ловушки - устанавливаются при прикосновение с поверхностью
