# Описание
Во время игрового процесса игрок может управлять персонажем и взаимодействовать с окружением.

> Во время игрового процесса могут быть включены [[Сюжетная вставка|сюжетные вставки]].

# Составляющие игрового процесса
Весь игровой процесс можно разделить на 3 основных составляющие:
- [[Изучение локации]]
- [[Подготовка локации]]
- [[Выполнение задания|Задание миссии]]
