# `= this.file.name`
[[Персонаж игрока]] может взаимодействовать с [[Лазелка]].

Оказавшись рядом с [[Лазелка]] и подпрыгнув [[Персонаж игрока]] цепляется за неё (персонаж не обязательно обязан стоять на земле).


Перемещение по [[Лазелке]] реализуется свободным движением параллельно стене как паук.
