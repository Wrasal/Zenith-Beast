# `= this.file.name`
Карабканье в игре представлено в виде взаимодействия персонажа со специальными объектами - [[Скаладром]].

[[Персонаж игрока]] может:
- Запрыгнуть на [[Скаладром]] - (Персонаж во время прыжка, столкнувшись со [[Скаладром|скаладромом]], если зажата кнопка space, зацепляется за него переходя в режим карабканья)
- Карабкаться - (W/S двигается по вертикали, A/D двигается по горизонтали, комбинируется для перемещения по диагоналям)
- Спрыгнуть - (CTRL)
- Отпрыгнуть - (SPACE + W/A/S/D персонаж отпрыгивает от стены W - вверх, A/D - влево/вправо, S - назад за спину)
- Осматриваться - [[Камера]] работает стандартно в этом режиме.
- <span style="background:#0099995E">Стрелять</span>

<span style="background:rgba(146, 84, 222, 0.37)">### Взаимодействие</span>
<span style="background:rgba(146, 84, 222, 0.37)">Игрок может взаимодействовать со стеной:</span>
<span style="background:rgba(146, 84, 222, 0.37)">- начать карабканье</span>
<span style="background:rgba(146, 84, 222, 0.37)">	- Во время падения </span>
<span style="background:rgba(146, 84, 222, 0.37)">	- Вблизи нажать кнопку взаимодействия</span>
<span style="background:rgba(146, 84, 222, 0.37)">- закончить карабканье</span>
<span style="background:rgba(146, 84, 222, 0.37)">	- Отпрыгнуть от стены</span>
<span style="background:rgba(146, 84, 222, 0.37)">	- Доползти до конца маршрута</span>

<span style="background:rgba(146, 84, 222, 0.37)">### Особенности</span>
<span style="background:rgba(146, 84, 222, 0.37)">**Управление** WASD не зависимо от камеры</span>
