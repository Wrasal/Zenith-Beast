# Интерактивность
Интерактивность - активация предмета с помощью гаджета
Каждый элемент можно сделать интерактивным. Когда элемент интерактивен он скрыт, сломан, не активен ...

Взаимодействовать можно с помощью:
- Оружия (выстрелить)
- Гаджетами (таких гаджетов ещё нет)
- Животными (помощь животного)

## Предложение
Для первого акта подойдут интерактивная растительность
- Скрытые проходы в корнях (для новых стелс путей)
- Акти паркурных путей для выгодных(тактичных) позициях
