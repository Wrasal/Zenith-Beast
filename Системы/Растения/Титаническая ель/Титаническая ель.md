# Титанические ели
Огромные круглые стволы простирающиеся высоко вверх. Формируют своей листвой подобие зеленой крыши.

Могут использоваться как элемент вертикального геймплея. (прыгать по веткам...)

## Корни
Корни титанических деревьев. Крупные словно целые стены, преграждают перемещение по поверхности.
